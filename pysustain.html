目次

1.Anacondaのインストール

Anacondaは次のページからインストールできます。 https://www.anaconda.com/download 好きなフォルダを選択してインストールしてください。

2.Visual Studio Build toolのインストール

WindowsのpythonではLinuxのようにC++コンパイラがOSにないため、この後の作業でErrorがでてしまいます。それを解決するのがMicrosoft C++ Build Toolsです。 ダウンロード ダウンロードしたファイルを開くと、いろいろな項目を選択してVisual Studio Build toolをインストールできますが、今回はC++に関するものだけで大丈夫です。

3.環境設定

anacondaで仮想環境を作っていきます。

仮想環境の構築

anacondaのインストールによって、Anaconda prompt(Miniconda)がアプリの中に入ると思います。これを開き、
conda create env --name sustain_env pythin=3.7
conda activate sustain_env
conda install numpy
これをそれぞれ実行します。

PySuStainのダウンロード

PySuStaInのGithubページ https://github.com/ucl-pond/pySuStaIn に行き、任意のフォルダにダウンロードします。Gitを使って、クローンを作っていただいても大丈夫です。
PySuStaInをダウンロードしたら、Minicondaで
cd pySuStaIn
pip install . (もしくは　pip install -e .)
と入力します。
